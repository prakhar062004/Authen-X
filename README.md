# Authen-X

Authen-X is an AI-powered web application that helps users verify the authenticity of information from news articles, images, and text. It analyzes content and provides insights to identify potential misinformation.

## Features

- News URL Verification
- Image Authenticity Analysis
- Text Fact Checking
- AI-Powered Content Validation
- User-Friendly Interface
- Real-Time Analysis Results

## Tech Stack

### Frontend
- React.js
- JavaScript
- CSS/Tailwind CSS

### Backend
- Node.js
- Express.js

### AI Integration
- External AI APIs
